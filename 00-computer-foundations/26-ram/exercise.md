# Question 3

### Explain the difference between RAM and SSD without using the words "temporary" or "permanent." Focus on how they behave.

RAM only stores data that are active. It is volatile and once electricity is off, it loses all the data. SSD however is non volatile, can store data that for a longer period of time even if they are not active and does not depend on power or electricity.
