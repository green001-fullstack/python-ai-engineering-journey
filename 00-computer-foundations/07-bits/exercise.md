# Question 1

### What is a bit?

A bit represents one binary digit and can hold one of two possible values: 0 or 1.

# Question 2

### Why can one bit only represent two possible values?

One bit can only represent two possible state, either yes or no, no in between.

# Question 5 (Critical Thinking)

Complete this sentence:

"A bit is not valuable because it stores a lot of information. A bit is valuable because..."

A bit is valuable because many bits can be combined to represent increasingly complex information. A bit is valuable because it is the smallest building block from which all digital information is created.
