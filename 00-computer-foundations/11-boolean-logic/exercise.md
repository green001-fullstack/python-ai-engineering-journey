# Question 1

### What is a Boolean?

A Boolean is a data type that can hold one of two values: True or False. It represents the result of a logical question.

# Question 2

### How is a Boolean similar to a bit, and how is it different?

Bit	Boolean
Stores one binary digit (0 or 1)	Represents logical truth (True or False)

The important idea is:

A bit is about storage.

A Boolean is about meaning.

Just like we learned:

Bytes store data.

Character encoding gives that data meaning.

We're seeing the same pattern again.
