# Responsibilities of the different components of CPU

| Responsibility         | Component      |
| ---------------------- | -------------- |
| Coordinate             | ✅ Control Unit |
| Calculate              | ✅ ALU          |
| Store temporary values | ✅ Registers    |
| Synchronize            | ✅ Clock        |
