# Question 5 (Critical Thinking)

### Imagine you're designing a calculator. If it forgets every answer immediately after calculating it, what kinds of calculations would become impossible or very difficult?

Calculations that depend on previous results become impossible. For example, repeated addition, multiplication, factorials, averages, Fibonacci numbers, and many loop-based algorithms all require memory because each step depends on the result of the previous step.
