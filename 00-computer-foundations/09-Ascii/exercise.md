# Question 1

### In your own words, what is ASCII?

ASCII is a standard way of representing characters by mapping characters to numbers.

# Question 2

### Why was ASCII created?

ASCII was created so different computers could use the same representation for characters.

# Question 4

### True or False? The computer stores the actual letter "A" in memory. Explain your answer.

False. The computer stores the letter A in binary form (0s and 1s). These numbers represent hardware behaviours (transistors).
