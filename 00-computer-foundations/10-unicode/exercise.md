# Question 1

### Why wasn't ASCII enough for modern computing?

Ascii can only represent a maximum of 128 characters, and these characters are majorly English alphabets or characters. We need to be able to represent non English characters and symbols. This is what gave birth to Unicode which is a universal character standard that assigns a unique code point to almost every character used by humans.
