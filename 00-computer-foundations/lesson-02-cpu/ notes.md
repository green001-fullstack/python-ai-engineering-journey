# What is a CPU?
CPU is a physical chip that helps to fetch, decode and execute machine instructions that helps to run our programs.

# Why can't the CPU execute Python code directly?
Because it does not understand python codes. Python is a high-level language designed for humans.

The CPU only understands machine instructions, which are binary patterns.

So the Python Interpreter acts as a translator.

Think of it like this.

You speak English.

Someone speaks Japanese.

Neither of you understands the other.

A translator stands between both of you.

Exactly what Python Interpreter does.

# Suppose your computer has: RAM, SSD, Keyboard, Monitor but no CPU. What happens when you press the power button, and why?
The computer will not start because there is no CPU available to execute any instructions, including the startup instructions stored in firmware.
