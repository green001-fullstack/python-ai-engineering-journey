# Question 1

### What is a byte?

A byte is a group of 8 bits

# Question 2

### Why did 8 bits become the standard size for a byte?

Because it is large enough to represent all lowercase letters, uppercase letters and punctuation. It has 256 possibilities that is large enough to represent those characters. It was also a good balance between: Memory efficiency (not wasting space) Flexibility (256 combinations) and Hardware design (reasonable for the technology of the time)

# Question 3

### How many different patterns can one byte represent?

256, One bit has 2 possibilities. Every additional bit doubles the possibilities.
Therefore:
1 bit → 2
2 bits → 4
3 bits → 8
...
8 bits → 256

# Question 4

### True or False? A byte always represents one letter

False. It depends on interpretation. Data has no inherent meaning.

# Question 5 (Critical Thinking)

### Complete this sentence: "A byte has no meaning by itself because..."
