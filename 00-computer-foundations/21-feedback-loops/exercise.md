# Question 1

### In your own words, what is a feedback loop?

A feedback loop is when the output of a circuit is fed back into the same circuit as a future input.

# Question 2

### What is the main difference between a normal logic circuit and a circuit with feedback?

A normal logic circuit flows in one direction while feedback has a repetition component that makes it non linear but somewhat circular.

# Question 3

### Why can feedback allow a circuit to remember something even after the original input disappears?

Because the input is the result of the previous operation. The circuit is effectively helping itself remember.
