# What is machine code?

Machine code is the binary instruction set that a specific CPU architecture can execute directly.

# Question 4

### What is an opcode, and what role does it play during the decode stage of the fetch–decode–execute cycle?

An opcode (Operation Code) is the part of a machine instruction that tells the CPU what operation to perform.

# Question 5 (Critical Thinking)

### Imagine a world where assembly language was never invented. Programmers would have to write programs directly in binary machine code. What problems do you think this would cause for software development?

Imagine Microsoft Word.

It contains millions of lines of code.

Now imagine millions of lines of this:

110010101001011010001001
101010001010010010101100
111010001001010101010001

Now imagine:

finding one bug,
adding one feature,
reviewing another programmer's work.

Software engineering would almost stop existing.

Assembly was one of the greatest inventions in programming history because it separated human thinking from machine thinking.
