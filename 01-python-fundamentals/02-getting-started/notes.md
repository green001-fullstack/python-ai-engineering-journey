Python is a programming language specification. CPython is the most popular implementation of that specification.

There are several implementations of Python:

CPython (written mostly in C)
PyPy
Jython
IronPython

All implement the same language specification.

The python interpreter is the software engine(python VM) which helps to translate byte code into machine codes line by line
