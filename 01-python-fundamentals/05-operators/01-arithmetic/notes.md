# Question 1

### What does an arithmetic operator produce? Does it usually modify an existing immutable object or create a new one? Explain why.

Arithmetic operators on immutable built-in types (int, float, bool, str) produce new objects instead of modifying existing ones.

# Question 2

### Why does this work? "Hello " + "World" even though addition is a mathematical operation.

The operators are interpreted differently for different data types.

# Question 3

### What is operator overloading?

The same operator can perform different actions depending on the types of the objects involved.
